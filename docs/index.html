Moonshine

A skill for distilling interactive technical explanations from AI-generated complexity.

AI tools generate code, systems, and documentation faster than teams can understand them. Moonshine turns that complexity into interactive explanations where readers can explore, test, and build intuition at their own pace.

Inspired by Distill.pub, these aren't summaries. They're decompositions: finding the right visualization, the right sequence of ideas, the right interaction that turns opacity into understanding.

Seeing the Parts

A complex waveform is hard to interpret whole. Fourier analysis decomposes it into sine waves. Toggle components to see how they combine.

You could read a paragraph explaining that low frequencies carry overall shape while high frequencies add detail. Or you could turn off the high frequencies yourself and watch the waveform smooth out. The second builds intuition the first cannot.

Explorations

Nine collections of interactive explanations, each built with moonshine. Over 195 articles across visualization technique, seismology, music data, machine learning, bioinformatics, complex systems, multidimensional geometry, and the system that builds them.

How It Works

Moonshine is a skill for AI coding assistants. It guides a structured process: story discovery, interaction design, scaffolding, and iterative building. It asks questions about what you're explaining and who the audience is before writing any code.

Manual install

git clone --depth 1 https://github.com/enjalot/moonshine.git /tmp/moonshine
cp -r /tmp/moonshine/plugins/moonshine ~/.claude/skills/moonshine
rm -rf /tmp/moonshine

The skill includes built-in D3 visualization patterns (VISUALS.md) covering chart types, interaction recipes, linked views, brushing, scroll-driven narrative, and the moonshine style foundation. Self-contained HTML, vanilla JS, D3 v7. No build tools, no frameworks.

Standing On

Distill.pub articulated why distillation matters in Research Debt. Mike Bostock created D3.js and showed how visualization and narrative work together. Bret Victor imagined explorable explanations. Nicky Case, Bartosz Ciechanowski, and The Pudding proved the format works across subjects.
