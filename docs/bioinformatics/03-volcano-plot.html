Volcano Plot

The most iconic chart in bioinformatics: fold change meets statistical significance to reveal which genes truly matter.

The MA plot shows which genes changed, but not whether the change is statistically reliable. A gene might show a 4-fold change by chance if the measurements are noisy. The volcano plot adds a second filter: the p-value from a statistical test.

The axes are:

  • Horizontal: log2(fold change) — how much the gene changed
  • Vertical: -log10(p-value) — how confident we are in the change

The name comes from the shape: most genes cluster at the bottom center (small change, not significant), while a few shoot up and outward like a volcanic eruption. Those are the genes of interest.

The Classic View

Each dot is one gene. Drag the threshold lines to define what counts as "significant" and "large enough change." Genes in the upper corners pass both filters.

Figure 1. Volcano plot with adjustable thresholds. The vertical dashed lines mark the fold-change cutoff; the horizontal dashed line marks the significance cutoff. Genes in the upper-left and upper-right quadrants are differentially expressed.

The four quadrants of a volcano plot: upper-left (significantly downregulated), upper-right (significantly upregulated), lower-left/right (large change but not significant — often low-expression genes with noisy estimates).

Why Two Thresholds?

Using fold change alone would include noisy genes. Using p-value alone would include genes with tiny but real changes (a gene that consistently changes by 1% might be statistically significant in a large study, but biologically meaningless). The volcano plot enforces both: the change must be large and reliable.

Figure 2. Left: fold-change filter only (false positives from noisy low-expression genes). Center: p-value filter only (captures tiny but real changes). Right: both filters (the volcano approach). The rightmost panel selects genes that are both large in effect and statistically confident.

Labeling the Top Hits

In practice, bioinformaticians label the most interesting genes — those farthest into the significant corners. Hover to see gene names, or adjust the top-N slider to label more.

The genes that survive both thresholds become a gene list. That list feeds into downstream analyses: pathway enrichment, protein interaction networks, or the survival analysis that asks whether these genes predict patient outcomes.
