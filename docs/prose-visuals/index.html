Prose Visuals: Algorithms & ML

Twelve interactive explanations of algorithms and machine learning techniques, each with multiple interactive figures. Built with moonshine's prose-visual approach: visuals described at the intent level, compiled to code.

These explanations demonstrate what happens when you describe visuals as prose rather than writing rendering code directly. Each explanation has four interactive figures that go beyond standard charts into animated algorithms, state machines, linked parameter explorers, and live simulations.

The concepts span optimization, probabilistic reasoning, neural networks, and unsupervised learning.

Part of the moonshine project. Built with PROSE-VISUALS.md.
