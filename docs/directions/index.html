Future Directions

24 interactive explorations of where d3-power-tools could go, especially in ways that support building interactive technical explanations with moonshine.

These directions span authoring workflows, data infrastructure, AI automation, accessibility, output platforms, and evaluation. Each is a self-contained interactive explanation that demonstrates the concept it proposes. They range from near-term additions (Observable Plot bridge, print export) to research frontiers (explanation effectiveness, skill ecology).

The explorations are organized into six categories. Some are practical extensions that could ship soon. Others are open questions about how visualization tools should work, presented as interactive arguments rather than roadmap items.
