Autoresearch: Thirteen Angles

How d3-power-tools uses autonomous AI loops to iterate on D3 visualizations. Each explanation looks at the same system from a different angle.

Karpathy's autoresearch is a simple idea: let an AI modify code, check if the result improved, keep or discard, and loop. d3-power-tools applies this pattern to D3 visualizations, using four quality dimensions and a composite score to decide what survives.

These thirteen explanations explore the system from different angles. The first seven explain how autoresearch works. The last six explore how to make it better using vision models, multi-objective optimization, smarter proposers, context compression, ensemble auditing, and cross-block transfer learning.
