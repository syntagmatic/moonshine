The Golay Code

4,096 binary words in 24 dimensions. A perfect code, a Steiner system, and the seed of the Monster.

An error-correcting code is a collection of binary strings chosen so that no two are too similar. If noise corrupts a few bits during transmission, the receiver can detect the corruption and recover the original string by finding the nearest codeword. The geometry is Hamming space: , the set of all binary strings of length , where distance counts how many positions differ between two strings.

In 1949, Marcel Golay discovered a code of extraordinary efficiency. Take and place 4,096 codewords so that any two differ in at least 8 positions. This is the extended binary Golay code, with parameters : length 24, dimension 12, minimum distance 8. It corrects any combination of up to 3 errors in a block of 24 bits.

What makes the Golay code remarkable is not just its error-correcting power but the depth of its mathematical connections. Its 759 weight-8 codewords form the unique Steiner system . Its automorphism group is the Mathieu group , a sporadic simple group of order 244,823,040. And through Construction A, it gives rise to the Leech lattice, which leads to the Conway groups and ultimately to the Monster.

Weight Distribution

The Golay code is self-dual: it equals its own dual code. It is also doubly even: every codeword has a weight divisible by 4. These two algebraic constraints force the weight distribution into a remarkably sparse shape. Out of the 25 possible weight values (0 through 24), only five occur.

Figure 2. Weight distribution of the Golay code. Only weights 0, 8, 12, 16, and 24 appear, with counts 1, 759, 2576, 759, and 1. The symmetry around weight 12 reflects the complement operation: if v is a codeword, so is v + 1.

Why only these weights? The code is doubly even, so all weights are divisible by 4. The minimum distance is 8, ruling out weight 4. MacWilliams identities (which relate a code's weight enumerator to its dual's) combined with self-duality then force the exact counts. The weight enumerator polynomial is .

The 759 codewords of weight 8 are called octads. They form a combinatorial design of profound significance. The 2,576 weight-12 words are dodecads, each the symmetric difference of two octads. The 759 weight-16 words are complements of the octads. And the two extreme words (all-zeros, all-ones) sit alone at the poles.

The Steiner System

The 759 octads form a Steiner system : a collection of 8-element subsets of a 24-element set such that every 5-element subset is contained in exactly one octad. This is the unique such system, up to relabeling of the 24 positions. Its existence was proved by Ernst Witt in 1938.

Only two non-trivial perfect binary codes exist. The binary Hamming code corrects 1 error, and the binary Golay code corrects 3 errors. Together with the trivial repetition codes, these are the only perfect binary codes. This was proved independently by van Lint and Tietäväinen in 1973.

From Golay to Monster

The Golay code is the first link in a chain of constructions that reaches the largest sporadic simple group. Each step passes symmetry upward:

The Mathieu group is the automorphism group of the Golay code: the set of all permutations of the 24 coordinates that map codewords to codewords. It has order and acts 5-transitively on the coordinates, meaning any ordered 5-tuple can be mapped to any other. It is one of the five Mathieu groups, all sporadic simple groups discovered by Emile Mathieu in the 1860s and 1870s.

Construction A builds a lattice from a code: take all vectors in that reduce modulo 2 to a Golay codeword, then scale by . The result is the Leech lattice, the unique even unimodular lattice in 24 dimensions with no roots. Its automorphism group (Conway's group) contains the Conway groups , , as subgroups.

The sporadic landscape. Of the 26 sporadic simple groups, 20 are subquotients of the Monster (the "happy family"). The Mathieu groups form the first generation. The Golay code is the combinatorial seed from which much of this exceptional structure grows. Article 27 will explore error-correcting codes more broadly; here we see why this particular code is singular.

The Golay code is an existence proof that extreme efficiency and deep symmetry can coincide. No other binary code achieves perfect error correction in such a rich algebraic setting. It is a single object that belongs simultaneously to coding theory, combinatorics, group theory, and lattice geometry, and it connects all four through constructions that would seem improbable if they were not provably true.
